Color changes can transform a simple project into a stunning work of art. Learning to change colors seamlessly is an essential skill for any crocheter.

## The Basic Color Change

The key to a clean color change is completing the last yarn over of your final stitch with the new color.

### For Single Crochet

1. Insert hook into next stitch
2. Yarn over and pull up a loop (2 loops on hook)
3. Drop the old color
4. Pick up the new color
5. Yarn over with new color and pull through both loops

### For Double Crochet

1. Work the dc until 2 loops remain on the hook
2. Drop the old color
3. Yarn over with new color
4. Pull through both loops

## Carrying Yarn vs. Cutting

### When to Carry

- Changing colors every 1-2 rows
- Stripes that repeat frequently
- When yarn floats will be hidden

### When to Cut

- Random color placement
- Many rounds between color uses
- Visible work where floats would show

## Tips for Clean Color Changes

1. **Change on the last stitch** of the row before you need the new color
2. **Weave in ends as you go** to save time later
3. **Keep consistent tension** when picking up the new color
4. **Do not pull too tight** on the first stitch of a new color

## Managing Multiple Colors

When working with many colors:

- Use bobbins to keep yarns organized
- Work over the unused color to hide it
- Plan your color sequence before starting

Practice these techniques, and your colorwork will look professional!
